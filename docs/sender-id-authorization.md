# Sender ID Authorisation — "LumeX" (DND delivery)

Goal: register the alphanumeric Sender ID **LumeX** on all 4 Nigerian MNOs so
OTP/transactional SMS delivers to DND lines (e.g. MTN) and shows "LumeX".

Sendchamp requires, per network (4 letters total) + once:
1. The **Authorisation Letter** below, on your company letterhead, addressed to each MNO.
2. Your **CAC Certificate** attached.

One signatory is fine (two not mandatory).

---

## Fill these in once, then reuse across all 4 letters
- `[COMPANY LEGAL NAME]` — your CAC-registered entity name
- `[CAC NUMBER]` — RC number from the CAC certificate
- `[YEAR]` — year of incorporation
- `[NATURE OF BUSINESS]` — suggested: *operating a campus food-delivery and student-services platform*
- `[SIGNATORY NAME]` / `[TITLE]` (×1 or ×2)

---

## Letter (official Sendchamp format — one copy per MNO, on letterhead)

```
<<Company Letterhead>>

<Insert MNO Address — see the 4 blocks below>

Dear Sir/Ma,

TO WHOM IT MAY CONCERN

Please be informed that Sendchamp Limited is an authorized aggregator of SMS
traffic for [COMPANY LEGAL NAME].

[COMPANY LEGAL NAME] with [CAC NUMBER] was incorporated in [YEAR], with the
sole objective of [NATURE OF BUSINESS] in Nigeria.

Kindly register "LumeX" as an approved sender ID with <Insert MNO> in Nigeria
to ensure delivery of sms notifications to our client.

[COMPANY LEGAL NAME] will be sending Transactional messages with this sender ID
to our customers who have consented to receive messages to be sent via the
sender ID.

We hereby state that we will not use this platform to send unsolicited and
illegitimate SMSs.

We thank you for your support.

Sample message:
"LumeX: your verification code is 123456. It expires in 10 minutes. Never share
it with anyone."

Yours faithfully,


……………………………………                       ……………………………………
[SIGNATORY NAME 1]                       [SIGNATORY NAME 2 (optional)]
[TITLE 1]                                [TITLE 2 (optional)]
```

> In the line "Kindly register "LumeX" … with **<Insert MNO>**", put the matching
> network name (MTN / Airtel / Glo / 9mobile) for each copy.

## Checklist
- [ ] Fill the placeholders (legal name, CAC no., year, business, signatory)
- [ ] Produce 4 copies — one per MNO (swap the address block + the `<Insert MNO>` name)
- [ ] Each on letterhead, signed; attach the CAC certificate to each
- [ ] Send the package back to Sendchamp support
- [ ] Approval typically ~1–2 weeks per network

## Until approval lands
OTP already DELIVERS to DND/MTN numbers via Sendchamp's **DND route** (see
[[termii-to-sendchamp]]). MTN DND lines may show a generic sender name until
"LumeX" is approved; the code SMS already begins with "LumeX:" so it's
recognisable. No code change needed once approved.
